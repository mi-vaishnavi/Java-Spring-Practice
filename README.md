# Java-Spring-Practice

This repository contains my Spring and Spring Boot practice programs, covering the core concepts required for backend development. The goal of this repository is to build a strong foundation in Spring by implementing concepts step by step with hands-on examples.

## 📌 Topics Covered

### Spring Core
- Spring Project Setup
- Inversion of Control (IoC)
- Dependency Injection (DI)
- Bean Creation
- Spring Container
- `@Component`
- `@Autowired`
- Constructor Injection
- Field Injection
- Bean Scopes (Basics)

### Spring Boot
- Spring Boot Project Structure
- Auto Configuration
- Application Properties
- Running Spring Boot Applications

### REST APIs
- Creating REST Controllers
- HTTP Methods
  - GET
  - POST
  - PUT
  - DELETE
- Path Variables
- Request Parameters
- Request Body
- JSON Response

### CRUD Operations
- Create
- Read
- Update
- Delete
- Service Layer
- Controller Layer
- Model Classes
- In-Memory Data Storage
